## 📖 项目描述 | Project Description

**DreamCanvas** 是一个创新的创意故事生成引擎，利用先进的叙事算法和角色行为系统，自动构建富有深度的故事情节和角色互动。

**DreamCanvas** is an innovative creative story generation engine that uses advanced narrative algorithms and character behavior systems to automatically construct deep storylines and character interactions.

### 🎯 核心特性 | Core Features

| Feature | Description |
|---------|-------------|
| 🌟 **多宇宙设定** | 支持平行世界、跨维度叙事 |
| 👥 **角色行为系统** | 智能角色决策与互动引擎 |
| � **情节生成器** | 自动编织复杂故事线 |
| 🎭 **主题引擎** | 自由选择悬疑、浪漫、科幻等主题 |
| ⚡ **分支叙事** | 多结局非线性故事结构 |
| � **模块化架构** | 轻松扩展新的叙事规则 |

## 🚀 快速开始 | Quick Start

### 安装 | Installation

```bash
# 克隆项目
git clone https://github.com/your-repo/DreamCanvas.git
cd DreamCanvas

# 运行故事生成器
python3 main.py
```

### 基本使用 | Basic Usage

```python
from narrative_core import ThemeEngine, PlotActions
from worlds import ShadowMetropolis, StellarOdyssey
from characters import HeroCharacter, TricksterCharacter
from story_generator import StoryEngine, StoryPlot

# 创建世界
world = ShadowMetropolis()

# 创建角色
hero = HeroCharacter(name="Alex")
mentor = TricksterCharacter(name="Mysterious Guide")

# 创建故事引擎
story = StoryEngine(world, [hero, mentor], theme="mystery")

# 生成故事
narrative = story.generate_story(episodes=5)

print(f"Story: {narrative.title}")
print(f"Chapters: {narrative.chapters}")
```
